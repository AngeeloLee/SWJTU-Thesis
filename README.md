# SJWTU-Thesis

西南交通大学本科生毕业论文LaTeX模板（误）v0.1.0

写得渣，轻喷。目前只支持本科生的毕业论文，持续更新中。。。


## 编译

```
> xelatex paper     // 第一次编译
> bibtex paper      // 参考文献编译
> xelatex paper     // 第二次编译
> xelatex paper     // 最后一次编译
```

使用方式见paper.tex，遇到报错和警告请先跳过 :-)
